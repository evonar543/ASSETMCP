# Example Prompts

Use these with an MCP client that has ASSETMCP enabled.

```text
Find CC0 pixel art assets for a top-down horror roguelike. Prefer Kenney, OpenGameArt, and itch.io, but block unclear licenses.
```

```text
Search for low poly forest assets, download a safe CC0 pack, extract it, make a gallery, and generate credits.
```

```text
Create a Phaser prototype called Dungeon Lantern using these imported assets. Include movement, an enemy, HUD, menu, restart flow, README, credits, and asset manifest.
```

```text
Audit this project assets folder for license problems, suspicious files, missing credits, and missing manifest entries.
```

```text
Replace all unclear-license assets with CC0 alternatives from Kenney, ambientCG, Poly Haven, or Quaternius.
```

```text
Create a Godot 4 prototype using local user-provided assets in the ASSETMCP library. Generate CREDITS.md and ASSET_MANIFEST.json.
```
